# Nuvlabs Homebrew Tap

Homebrew tap for [ComplyKit](https://github.com/nuvlabs/complykit) — compliance-as-code for startups.

## Install

```bash
brew install nuvlabs/tap/complykit
```

## What is ComplyKit?

One command to scan your AWS, GCP, and GitHub against SOC2, HIPAA, and CIS benchmarks.

```bash
comply init    # 2-minute setup
comply scan    # scan your infrastructure
comply fix     # step-by-step remediation
comply serve   # web dashboard at localhost:8080
```

## Upgrade

```bash
brew upgrade nuvlabs/tap/complykit
```

## Uninstall

```bash
brew uninstall complykit
brew untap nuvlabs/tap
```
